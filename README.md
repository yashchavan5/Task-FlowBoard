# FlowBoard — Task Management App

A modern, production-grade Kanban task management app built with React. Features drag-and-drop, dark mode, filtering, and Framer Motion animations.

## Features
- Kanban Board with drag-and-drop (DnD Kit)
- Task CRUD with modal forms
- Search & filter by assignee/priority
- Dark mode toggle
- Deadline tracking with overdue warnings
- Priority tags (Low/Medium/High)
- Activity timeline
- Responsive design with collapsible sidebar
- Table view on Tasks page
- Toast notifications

## Tech Stack
React 18, TypeScript, Tailwind CSS, Framer Motion, Zustand, DnD Kit, date-fns, Sonner

## Getting Started
```bash
npm install
npm run dev
```
